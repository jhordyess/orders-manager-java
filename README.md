## Multi-container Docker app

Run the `docker-app.sh` file, it will create a new project named `orders-man-java` and run it; so you can visit [http://localhost:8080](http://localhost:8080).

```sh
bash docker-app.sh
```

Run for PowerShell:

```ps1
& ./docker-app.ps1
```
